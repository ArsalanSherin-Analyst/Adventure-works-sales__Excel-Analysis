# 🚴 AdventureWorks Sales Dashboard

## 📊 Overview

This project presents an **interactive Sales Dashboard** built using AdventureWorks data to analyze revenue performance across different dimensions such as **time, region, product categories, and customer preferences**.

The goal of this project is to transform raw sales data into **actionable insights** that support better business decision-making.

---

## 📁 Dataset

The dataset is based on the **AdventureWorks sample database**, a widely used dataset for data analytics and business intelligence projects.

It includes:

* Sales transactions
* Product categories (Accessories, Bikes, Clothing, Components)
* Regional data (Europe, North America, Pacific)
* Time-based data (2002–2004)

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation & data visualization
* **Excel** – Initial data handling
* **SQL (MySQL)** – Data querying and transformation
* **Python (Pandas)** – Data cleaning and preprocessing

---

## 🔄 Project Workflow

1. **Data Extraction**

   * Imported dataset from structured sources

2. **Data Cleaning & Transformation**

   * Handled missing values
   * Standardized formats
   * Prepared data for analysis

3. **Exploratory Data Analysis (EDA)**

   * Identified revenue patterns
   * Compared regional and category performance

4. **Dashboard Development**

   * Built interactive visuals using Power BI
   * Added slicers for filtering (Year, Color, Region)

---

## 📈 Dashboard Features

* **Revenue Distribution (Pie Chart)**
  Shows contribution by regions (Europe, North America, Pacific)

* **Sales Trend Over Time (Line Chart)**
  Tracks monthly revenue trends across product categories

* **Category Performance (Bar Charts)**
  Compares sales of Accessories, Bikes, Clothing, and Components

* **Regional Analysis**
  Highlights which regions generate the most revenue

* **Interactive Filters**

  * Year selection (2002–2004)
  * Product color
  * Region-based filtering

---

## 🔍 Key Insights

* **Bikes** generate the highest revenue across all regions
* **North America** shows strong and consistent sales performance
* Sales fluctuate seasonally, with noticeable peaks in certain months
* Some product categories (like Components) contribute less but remain stable
